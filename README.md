# 拼豆图案生成器 (Pindou Pattern Generator)

一个专业的在线拼豆图案生成工具，能够将普通图片转换为拼豆制作图纸。

## ✨ 功能特点

- 🖼️ **图片上传**：支持多种格式图片上传
- 🎨 **参数调节**：可调节网格尺寸、颜色系统、颜色容差
- 🧩 **智能配色**：支持多种拼豆颜色系统
- 📊 **颜色统计**：详细统计所需拼豆数量
- 💾 **保存打印**：支持图纸保存和打印
- 📱 **响应式设计**：适配各种设备屏幕

## 🎯 使用流程

1. **第一步：上传图片** - 选择要转换的图片文件
2. **第二步：参数调节** - 设置网格尺寸和颜色选项
3. **第三步：生成图纸** - 获得完整的拼豆制作图纸

## 🛠️ 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **图片处理**：Canvas API
- **颜色算法**：RGB颜色空间匹配
- **部署**：Vercel

## 🎨 支持的颜色系统

- **标准拼豆**：常用拼豆颜色
- **Perler系统**：Perler品牌拼豆
  - Basic Colors (基础色)
  - Earth Tones (大地色系)
  - Fashion Colors (时尚色系)

## 📖 使用说明

### 图片上传
- 支持 JPG, PNG, GIF 等常见格式
- 建议图片尺寸适中，过大图片可能影响处理速度

### 参数设置
- **网格尺寸**：5-200像素，影响图纸精度和大小
- **颜色系统**：选择对应的拼豆颜色品牌
- **颜色容差**：控制颜色匹配的精确度

### 颜色统计
生成图纸后会显示：
- 每种颜色所需拼豆数量
- 颜色分布图表
- 总拼豆数量统计
